Privacy Policy for Manemark

Effective date: March 18, 2026

Manemark is a Chrome extension that lets users capture the visible text content of webpages and save it for later use as LLM context.

What Manemark Does

When a user clicks the Manemark button, the extension captures the visible text from the current webpage, along with basic metadata such as the page title, URL, and timestamp.

Information Manemark Collects

Manemark does not collect personal information, create user accounts, or track browsing activity in the background.

The extension only processes webpage content when the user explicitly activates it.

How Data Is Stored

Captured content is stored locally in the user’s browser using Chrome local storage. Manemark uses this storage so users can view, manage, delete, import, and export their saved Manemarks.

Data Sharing

Manemark does not sell, transmit, or share user data with third parties. Captured webpage content is not sent to external servers by the extension.

User Control

Users can delete saved Manemarks at any time from the extension interface. Users may also export their saved content manually.

Permissions Used

  • activeTab – used to access the currently active tab only when the user clicks the extension.
  • scripting – used to inject the content script needed to capture visible page text.
  • storage – used to save captured content and related metadata locally in the browser.

Changes to This Policy

This privacy policy may be updated from time to time. Any updates will be posted on this page.
